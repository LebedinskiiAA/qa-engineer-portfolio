# Test Plan — The Internet Web Forms

**Plan ID:** TI-TP-001  
**Author:** Aleksandr Lebedinskii  
**Date:** 18 July 2026

## Objective

Demonstrate focused web test design for authentication and standard controls, including positive, negative, state and usability checks.

## In scope

Form Authentication, Checkboxes, Dropdown, Dynamic Controls and validation messages. Desktop-browser functional and basic keyboard/semantic checks.

## Out of scope

Basic/Digest Auth browser dialogs, file upload/download, geolocation permission, load testing and security penetration testing.

## Exit criteria

All P0 cases executed, results recorded, and any observation contains reproducible steps plus expected/actual behavior.
